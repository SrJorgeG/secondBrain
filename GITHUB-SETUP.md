# Second Brain

## Configuración de GitHub

### 1. Crea el repo en GitHub:
- Ve a: https://github.com/new
- Nombre: second-brain
- Descripción: Second Brain personal - Trading y conocimiento
- Público o Privado (tu elección)
- NO inicialices con README (ya tenemos uno)

### 2. Ejecuta en tu terminal:

cd ~/openclaw-workspace/second-brain
git remote add origin https://github.com/TU-USUARIO/second-brain.git
git branch -M main
git push -u origin main
```

### 3. Autenticación:
Si te pide login, usa tu usuario de GitHub y **token personal** (no contraseña):
- Crea token en: https://github.com/settings/tokens
- Selecciona scope: `repo`
- Usa ese token como contraseña
