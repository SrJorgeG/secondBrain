# 🧠 Second Brain - OpenClaw Workspace

Sistema de gestión del conocimiento personal organizado según la estructura del servidor.

## 📁 Estructura (Método PARA)

```
second-brain/
├── 00-Inbox/              📥 Captura rápida (procesar regularmente)
├── 01-Areas/              🗂️ Áreas de responsabilidad (canales del servidor)
│   ├── OpenClaw/          
│   ├── Trading-Inversiones/
│   └── General-Voz/
├── 02-Resources/          📚 Recursos de referencia
│   ├── Documentacion/
│   ├── Guides/
│   └── Templates/
├── 03-Projects/           🎯 Proyectos con fecha de entrega
├── 04-Archive/            📦 Notas antiguas/inactivas
└── 05-Daily/              📅 Notas diarias (journaling)
    └── 2025/
```

## 🔄 Flujo de trabajo

1. **Capturar** → Todo va a `00-Inbox/` primero
2. **Procesar** → Mover a área correspondiente o crear recurso
3. **Desarrollar** → Conectar notas con links `[[nota]]`
4. **Revisar** → Revisión semanal de inbox y áreas
