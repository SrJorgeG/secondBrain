## ⚙️ Configuración

### Workspace
- Ubicación: `~/openclaw-workspace/`
- Estructura: Second Brain, docs, projects
- Obsidian vault: `~/openclaw-workspace/docs/` y `~/openclaw-workspace/second-brain/`

### Herramientas Configuradas
- **Obsidian**: Vault de documentación
- **Comando rápido**: `openclaw-docs` - Abre Obsidian con el vault

## 🔧 Comandos Útiles

```bash
# Abrir Obsidian con vault de docs
openclaw-docs

# Ver estructura del workspace
ls -la ~/openclaw-workspace/

# Ver segundo cerebro
ls -la ~/openclaw-workspace/second-brain/
```
